# shoppe
🛍️ Shopee Static Websit
Giới thiệu
Đây là dự án website tĩnh mô phỏng giao diện Shopee được xây dựng hoàn toàn bằng HTML và CSS. Mục tiêu của dự án là rèn luyện kỹ năng thiết kế giao diện web thương mại điện tử, bố cục sản phẩm, và responsive design.
Tính năng chính
- Trang chủ với banner, danh mục sản phẩm và gợi ý mua sắm.
- Danh mục sản phẩm hiển thị theo lưới, dễ dàng tùy chỉnh.
- Thanh điều hướng và footer mô phỏng giống Shopee.
- Responsive design: hiển thị tốt trên cả máy tính và điện thoại.
- CSS styling: sử dụng flexbox, grid, và hiệu ứng hover để tăng trải nghiệm người dùng.
